# NOVA RETAIL — Loss Prevention Diagnostic

## Forensic Data Analysis for Retail Shrinkage Detection

End-to-end forensic analytics project simulating a real-world
loss prevention engagement for a 187-store retail chain
experiencing $4.7M in unexplained inventory shrinkage.

### Tech Stack
- Python (pandas, numpy, scikit-learn, networkx)
- SQL (SQLite for forensic queries)
- Plotly / D3.js (interactive visualizations)
- Reveal.js (HTML presentation framework)
